This project uses third party API services, and API Keys for this project to use are not distributed with this project. Follow these steps to set up the project:

1. Contact the developer to receive a copy of the API keys used for this project
2. Open the Xcode project and create a new target using the "VideoGameCollection" project
3. Open the "Edit Schemes" page, and click on Run->Arguments
4. Add the following environment variables with these key names, plus the API keys as the values that were distributed privately to you:
- Key: rawgAPIKey, value: insert-API-key-here
- Key: barcodeLookupAPIKey, value: insert-API-key-here
- Key: bingSearchAPI, value: insert-API-key-here

API keys should never be distributed without the written authorization of the developer
